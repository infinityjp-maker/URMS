# 01 — 製品概要（1 ページ）



> **version:** 0.2 · **User Go:** 2026-07-08



## URMS とは（あなた向け）



**URMS は「生活と仕事の状態を、ハブ + 機能画面で見せるデスクトップアプリ」** です。



| 比喩 | 意味 |

|------|------|

| **ハブ（窓）** | 朝・昼・夕・夜で見せる情報量が変わる入口 |

| **機能画面** | 天気詳細 · カレンダー · 交通 · 資産 … タップで詳細へ |

| **台帳（裏方）** | 予定 · 地点 · タスク · 天気などをデータベースに保存 |

| **完了 → 次へ** | 今日のタスクを終えたとき、次のタスクへ進む操作 |



Resource 一覧のような **管理画面** は Phase 3 の名残であり、**5173 / 5180 は削除済** · **製品の中心ではありません**。



## あなたが毎日使うもの



1. **URMS ハブ** — 時刻 · 状態 · 天気 · 予定 · タスク · AI からのひとこと  

2. **機能画面** — 天気詳細（S2 済）· カレンダー等（S3 以降）  

3. **「完了 → 次へ」** — 日次ループ（1 日の区切り）  

4. （将来）Windows にインストールした **デスクトップアプリ** — 今の 1420 と同じ見た目



## あなたが毎日使わないもの



| もの | 理由 |

|------|------|

| 旧 5173 / 5180 | **削除済** — 見なくてよい |

| Cursor 書き戻し | 開発インフラ（develop モード時のみ · PM/AI 用） |
