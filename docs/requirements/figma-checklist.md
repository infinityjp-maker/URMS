# URMS MVP — Figma 作業チェックリスト（User 用）

> Figma アカウント作成済みの方は、この順番で進めてください。

## Step 1 — ファイル作成（5 分）

1. https://www.figma.com/ を開く
2. 左上 **+ Design file**
3. ファイル名を **URMS MVP** に変更

## Step 2 — テンプレート 1 枚（15 分）

1. ツール **Frame（F）** → サイズ **1280 × 800**
2. 名前: `_Template / operate`
3. 上: ヘッダ（URMS + plan/operate/audit ボタン）
4. 左: ナビ（ホーム / Resource / Knowledge）
5. 中央: 空白（各画面の内容）

## Step 3 — 優先 4 枚（30 分）

| 複製して名前変更 | 中身 |
|------------------|------|
| `SCR-02 Resource List / operate` | 表 + [+ 新規作成] |
| `SCR-03 Resource Detail / operate` | 詳細 + [編集] |
| `SCR-04 Resource Create / operate` | フォーム |
| `SCR-07 Audit Log / audit` | 監査テーブル（audit Mode 選択） |

## Step 4 — 残り 5 枚（任意・後追い可）

SCR-01, SCR-05, SCR-06, SCR-08, SCR-09（前回 PM 指示参照）

## Step 5 — 共有

1. 右上 **Share** → **Anyone with the link** → **can view**
2. URL をコピー → [ui-design-links.md](./ui-design-links.md) の Figma URL に貼る
3. PM に「Figma URL 記載しました」と連絡

**Developer 実装は ui-requirements 準拠で進行中。Figma URL は後から反映可能。**
