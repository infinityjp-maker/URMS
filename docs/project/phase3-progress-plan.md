## 30秒で把握（テキスト版）

**URMS** = サーバーなどを管理する Web アプリを、**9 段階** で作っている。

- **終わった:** ① 土台 → ② 保存 → ③ 裏側 → ④ 画面
- **いま:** ⑤ 作業状況の画面（コード完成 · Git 保存待ち）
- **次:** ⑥ AI → ⑦ 拡張 → ⑧ テスト → ⑨ 公開準備

**あなた:** 普段は何も不要。画面を見たいときだけ Canvas の「画面の開き方」を参照。
