## 概要

`v1.3.0` 以降の **Cursor 書き戻し v1.4（食い違い検知 · 限定 merge 拡張）** を反映したマイナーリリース。

## 主な変更（v1.3.0 → v1.4.0）

| 領域 | 内容 |
|------|------|
| 書き戻し v1.4 | `contentHash` 不一致 → **conflict** 報告 · **上書き禁止** |
| Context 書き戻し | `exportContentHash` baseline · 競合時 skip |
| AI Team 書き戻し | `## URMS Export` に **Status** · **Owner** 行を追加（限定ゾーン） |
| API レポート | `CursorCombinedExportReport.conflicts` 集計 |
| UI | 書き戻し結果に「食い違い N 件（上書きしませんでした）」表示 |
| DB | `context_snapshots.export_content_hash` 列追加 |
