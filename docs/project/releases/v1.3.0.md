> **resource_type:** knowledge

## 概要

`v1.2.0` 以降の **Cursor export v3（本文 merge 限定ゾーン）** · **Desktop 白画面対策** を反映したマイナーリリース。

## 主な変更（v1.2.0 → v1.3.0）

| 領域 | 内容 |
|------|------|
| Cursor export v3 | AI Team `## URMS Export` ↔ `metadata.urmsSummary` 双方向 |
| Context export v3 | `ssotLinks` → `.cursor/context/` リンク節書戻し |
| AI Team sync | sync 時に URMS Export Summary を metadata へ取り込み |
| Desktop | ErrorBoundary · Vite optimizeDeps ガード — 白画面再発防止 |
| dev 運用 | plugin 再ビルド手順の安定化（v1.2.0 継続） |
