# URMS v1.2.0 リリースノート

> **resource_type:** knowledge  
> **resource_id:** knowledge:release-v1.2.0  
> **date:** 2026-07-08  
> **tag:** `v1.2.0`

## 概要

`v1.1.0` 以降の **Cursor 双方向 export（AI Team H1 + Context SSOT）** と **dev 起動安定化** を反映したマイナーリリース。

## 主な変更（v1.1.0 → v1.2.0）

| 領域 | 内容 |
|------|------|
| Cursor export v1 | `POST .../export` — AI Team Resource.name → 正本 Markdown H1 |
| Cursor export v2 | Context DB summary → `.cursor/context/`（Task · Phase · サマリ） |
| Integration API | `CursorCombinedExportReport`（aiTeam + context） |
| develop UI | Web `/develop/integrations` · Desktop DevelopPanel — 書戻し |
| dev 運用 | `dev:prepare` に plugin ビルド追加 — API オフライン再発防止 |
